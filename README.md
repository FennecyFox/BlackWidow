# BlackWidow

Web crawler


## Install

### Ubuntu

``` bash
$ sudo apt-get install python-dev libxml2-dev libxslt1-dev
$ pip install -r requirements.txt
```

### Mac OS X

``` bash
$ pip install -r requirements.txt
```


## Usage

``` bash
$ scrapy crawl beautylegmm
$ scrapy crawl fancy
$ scrapy crawl pinterest
```
